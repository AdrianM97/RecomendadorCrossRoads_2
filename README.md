# RecomendadorCrossRoads
Recomendador desarrollado para el videojuego CrossRoads, durante la estancia en el GIR ECA-SIMM.

# Instalación
Diseñado para ejecutarse como un contenedor docker. 

Implementado en Python (Python 3.8) como un servicio web Flask. 

El servicio web corre en el puerto 5000. 

En `requirement.txt` se recogen las librerias necesarias. 

`Dockfile`es el archivo de configuración del docker.
